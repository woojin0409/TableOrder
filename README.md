# 🍽️TableOrder

기존 음식점 주문 방식은 직원 호출 및 수기 전달 과정으로 인해
주문 누락이나 대기 시간이 발생하는 문제가 있습니다.

본 프로젝트는 매장 내 태블릿을 활용하여
고객이 직접 메뉴를 주문하고, 주문 데이터가 서버를 통해
POS 시스템으로 실시간 전달되는 스마트 주문 시스템입니다.

✨ 주요 기능
📱 태블릿 기반 메뉴 주문 시스템
🔄 서버 기반 실시간 주문 데이터 전송
🖥️ POS 화면 실시간 주문 확인
🛒 주문 내역 및 상태 관리
📊 메뉴 및 주문 데이터 관리
⚡ 실시간 주문 처리 기능
